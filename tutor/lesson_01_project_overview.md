## 🎯 第一部分：项目背景与业务价值（15分钟）

### 1.1 为什么需要智能水坑检测？

**现实问题**：
- 高速公路雨天积水易引发"水漂"事故
- 传统人工巡检效率低、成本高、覆盖不全
- 缺乏科学的积水风险量化评估方法

**解决方案价值**：
- ✅ **自动感知**：无人机点云数据自动采集
- ✅ **精准识别**：基于物理模型的水膜推演
- ✅ **快速处治**：智能生成最优处治方案
- ✅ **成本节约**：相比传统铣刨可节约80%成本

### 1.2 项目核心技术亮点

| 技术领域 | 具体应用 | 创新点 |
|----------|----------|--------|
| **计算机视觉** | 连通域病害识别 | CV算法在工程领域的创新应用 |
| **流体力学** | 填洼算法水膜推演 | 物理约束的数值模拟 |
| **车辆动力学** | 江守一郎滑水公式 | 理论公式的工程化实现 |
| **概率统计** | 风险等级划分 | 对数尺度的风险量化 |
| **Web开发** | Streamlit交互界面 | 科学计算与用户体验结合 |

## 🔧 第二部分：技术架构深度解析（15分钟）

### 2.1 四大核心功能模块

#### 🗺️ 模块1：3D高精度路表重构
- **输入**：无人机点云数据（.h5格式）
- **处理**：异常值修复、多路段拼接、超分辨率插值
- **输出**：高精度3D地形可视化

#### 🌊 模块2：动态降雨流体推演  
- **输入**：3D地形 + 降雨参数
- **处理**：填洼算法、径流系数、边缘约束
- **输出**：动态水膜分布过程

#### 🚨 模块3：全域滑水风险评估
- **输入**：水膜厚度分布 + 车速分布
- **处理**：临界速度计算、概率风险评估
- **输出**：A-E五级风险热力图

#### 🛠️ 模块4：智能排水处治决策
- **输入**：高危区域信息
- **处理**：连通域分析、工艺匹配、成本核算
- **输出**：靶向处治方案 + 经济效益对比

## 🔍 第六部分：常见问题FAQ

### Q1：运行时提示"找不到sample_data.h5"怎么办？
**解决方案**：
- 检查 `assets` 文件夹是否存在 `sample_data.h5`
- 如果不存在，可以自己创建一个简单的测试文件：
```python
import h5py
import numpy as np

# 创建测试数据
test_data = np.random.rand(100, 50) * 0.1  # 100x50的随机高程数据

with h5py.File('assets/sample_data.h5', 'w') as f:
    group = f.create_group('road_segments')
    group.create_dataset('segment_001', data=test_data)
```

### Q2：Streamlit启动很慢怎么办？
**可能原因**：
- 网络问题（首次启动需要下载前端资源）
- 端口冲突
- 内存不足

**解决方案**：
```bash
# 清除Streamlit缓存
streamlit cache clear

# 使用不同端口
streamlit run app.py --server.port=8502
```

### Q3：如何在没有网络的情况下运行？
**离线运行准备**：
```bash
# 预先下载所有依赖
pip download -r requirements.txt -d ./offline_packages

# 离线安装
pip install --no-index --find-links ./offline_packages -r requirements.txt
```
