### 第二阶段：核心技术模块深度学习 (5-7天)

#### 2.1 数据处理与3D重建模块
**文件**：`app.py` 中的 `load_and_preprocess_h5()` 函数

**学习重点**：
- HDF5文件格式处理 (`h5py` 库)
- 点云数据预处理技术：
  - 异常值检测与修复（3σ原则）
  - 多路段拼接算法（重叠区域加权融合）
  - 数据标准化（百分位数裁剪）

**实践建议**：
```python
# 尝试修改预处理参数
max_std = 15.0  # 异常值阈值
overlap_rows = 8  # 重叠行数
```

#### 2.2 流体力学水膜推演引擎
**文件**：`app.py` 中的 `simulate_water_film_with_low_wall()` 函数

**学习重点**：
- **填洼算法原理**：基于局部最小值的水体填充
- **物理约束**：边缘挡水墙、径流系数
- **数值稳定性**：最大水位爬升步长控制

**核心算法理解**：
1. 找到所有局部洼地（8邻域最小值）
2. 按洼地深度分层填充
3. 洼地溢出时的水流分配
4. 迭代收敛条件

**扩展学习**：
- 对比其他水文模型（如D8算法）
- 研究不同降雨强度下的积水模式

#### 2.3 车辆动力学风险评估
**文件**：`modules/risk_assessment.py`

**学习重点**：
- **江守一郎公式**：临界滑水速度计算
  ```
  vc = 145.0 × (水膜厚度)^(-0.4)
  ```
- **概率风险模型**：正态分布车速 + 生存函数
- **风险等级划分**：基于数量级的概率阈值

**数学基础**：
- 正态分布的累积分布函数(CDF)
- 生存函数(Survival Function)：P(X > x) = 1 - CDF(x)

#### 2.4 计算机视觉病害识别
**文件**：`modules/treatment_decision.py`

**学习重点**：
- **连通域标记算法** (`scipy.ndimage.label`)
- **形态学处理**：3×3结构元素的连通性定义
- **特征提取**：面积、深度、位置等几何特征

**CV知识点**：
- 二值图像连通域分析
- 区域属性测量
- 噪声过滤（面积阈值0.2㎡）

### 常见问题排查
- **HDF5文件读取失败**：检查文件路径和权限
- **内存不足**：减小处理的数据量或增加虚拟内存
- **可视化异常**：检查数据范围和坐标系设置
- **算法收敛慢**：调整迭代步长和收敛阈值
